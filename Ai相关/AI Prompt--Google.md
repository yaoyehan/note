## 1. How to create effective promots? 
```ad-note
遵循下面五步！！！
- **Task**
  *Persona*-角色
  *format*-格式
- **Context**
- **References**
- **Evaluate**
- **Iterrate**
```
1. ==Task-任务==
任务可以从下面两方面着手：
-   *Persona*-角色
> 你可以指定Ai所扮演的角色，比如：有15年经验的主管，或者扮演客户和你的经理沟通。
-  *format*-格式
> 内容输出的格式，比如短语、表格等。
1. ==Context-上下文==
其实就是对上面Task的补充
> 比如：我需要Ai给我30岁以下生日礼物的想法
2. ==References-引用==
当让Ai输出一些抽象概念的时候，可以给Ai参考资料
3. ==Evaluate-评估==
确认当前Ai输出的是不是你想要的
4. ==Iterrate-迭代==
如果上一步的评估对Ai的结果不是很满意，可以不断调整提示词

## 2.四种迭代方式
1.重新审批提示词框架
> 确保你提供了足够的特异性在你的任务、上下文和参考中
eg：
![[1.提示词特异性.png]]
2. 分离你的提示缩短句子
> 这样做的好处是Ai可以每次处理一个单独的任务，而不用考虑多个句子之间的关系
![[2.缩短提示词.png]]
3. 尝试不同的表述方式或切换到类似的任务
> 从编写市场计划到写写一个故事,讲述这款产品如何融入我们目标客户群体的生活。
![[3.尝试切换不同的表述方式.png]]
4. 引入约束
> 引入约束可以使得Ai工具缩小其输出范围
## 3. 如何让Ai给我们生成图片
- Size - 大小
- Color - 颜色
- Position - 位置
- Aesthetic - 美感
eg：
```
为海报生成一张电吉他的图片。图片应采用摄影风格。吉他应该有闪耀或闪烁的效果,营造出一种兴奋感。吉他应位于前景,给人一种漂浮在天空中的感觉。
```
